An easy/quick subscribe widget

== Description ==
This plugin adds a widget that allows blogs to accept email addresses for subscriptions to their blog. It quickly adds a new email to the subscriber list in the Wordpress system if the email does not exist.

== Installation ==
Upload the easy subscribe folder and all of its contents into your plugins directory (/wp-content/plugins/ by default)
Open the WordPress admin panel and go to the \'Plugins\' page
Activate the new \'Easy Subscribe\' item in the list of plugins
